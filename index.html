Fork me on GitHub

github ribbon

Меня всегда бесили ленточки Fork me on github , выполненные в виде картинок. Поэтому я решил сделать свою ленточку для github на чистом сss. За одно и потренировался с новыми для меня css3 свойствами: transform и transform-origin.

how to use

Для того чтобы использовать ленточку на своем сайте необходимо:

  1. подгрузить стиль

    <link rel="stylesheet" type="text/css" href="ribbon.css" />

    или с использованием rawgithub

    <link rel="stylesheet" type="text/css" href="https://rawgithub.com/webcane/github-ribbon/master/ribbon.css" />

  2. и добавить код

    <div class="ribbon">
  <a id="orange" href="https://github.com/webcane/github-ribbon">Fork me on GitHub</a>
</div>

narrative

На данный момент отсутствует возможность разместить ленточку по разным углам экрана используя только стили.

В начальной версии был добавлен только один цвет ленточки - оранжевый. Остальные цвета возможно добавлю после.

Была идея еще поиграться со стилем оформления ленточки:

  • прошить как иголкой по краям
  • использовать :before и :after для того, чтобы срезать края ленточки под 45°, а не тупо прятать их.
  • добавить тайминг для ховера
